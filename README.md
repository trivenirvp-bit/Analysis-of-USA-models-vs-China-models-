# Project Title
# Comparative Analysis of Large Language USA-models-vs-China-models-

## Description
This project analyzes and compares different large language models based on performance, accuracy, and use cases.
Conducting a comparative analysis of various large language models (LLMs) from USA and China to determine their suitability across tasks like text generation, summarization, and question answering .Assess the performance, efficiency, and adaptability of each model (Llama, Qwen, Grok, DeepSeek, Gemini)
## Tools Used
- Python
- Machine learning(Deep Learning)
- NLP
